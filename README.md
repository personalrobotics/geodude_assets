# mj_geodude

MuJoCo software for Geodude.

## Setup

```
pip install .
```

## Development

[Ruff](https://docs.astral.sh/ruff/) is used for linting and formatting python code:
```bash
# Check for linter errors
ruff check .
# Apply lint fixes
ruff check --fix .

# Check for format errors
ruff format --diff .
# Apply format fixes
ruff format .
```
